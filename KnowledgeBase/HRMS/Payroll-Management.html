1 Overview

The Payroll Management page is where you configure and process employee compensation. This comprehensive module handles everything from defining salary components to generating payslips.

Salary Components

Define earnings (Basic, HRA, DA) and deductions (PF, ESI, Professional Tax) that make up an employee's salary.

Location Taxes

Configure location-specific taxes like Professional Tax with different rules for different offices.

Salary Structures

Create salary templates by combining components. Supports versioning for historical tracking.

Loans & Advances

Manage employee loans with interest calculations and automatic EMI deductions from salary.

Arrears

Handle salary arrears from promotions, appraisals, or corrections with automatic payroll inclusion.

Payroll Drafts

Create and review payroll drafts before finalizing. Edit individual employee entries as needed.

Finalized Runs

View completed payroll runs, download payslips, export bank files, and track payment status.

ℹ️
Setup Order: Configure in this order for best results: Salary Components → Location Taxes → Salary Structures → Assign to Employees → Process Payroll.

2 Payroll Workflow

Understanding the payroll processing workflow is essential for accurate and timely salary payments.

The Payroll Processing Cycle

Draft → Process → Approve → Paid

Workflow Stages

Stage Description Actions Available
Draft Initial payroll creation. System calculates salaries based on structures and attendance. Edit entries, add adjustments, delete draft
Process Review calculated amounts, make final adjustments, and verify totals. Edit entries, apply arrears, recalculate
Approve Payroll is finalized and locked. No more edits allowed. View payslips, download CSV, generate bank file
Paid Salaries have been disbursed to employee bank accounts. View payslips, download reports
⚠️
Important: Once a payroll is approved, it cannot be edited. Always verify all calculations before approving. If corrections are needed after approval, they must be handled in the next payroll cycle as arrears.

3 Salary Components

Salary components are the building blocks of employee compensation. They are divided into two categories: Earnings (additions to salary) and Deductions (subtractions from salary).

Components Overview

Salary Components tab showing earnings and deductions tables with filter dropdown

Understanding Component Types

Type Description Examples
Earning Amounts added to the employee's gross salary Basic Salary, HRA, DA, Conveyance, Medical Allowance
Deduction Amounts subtracted from gross salary PF, ESI, Professional Tax, Income Tax, Loan EMI

Adding a New Component

Click the "Add Component" button to create a new salary component.

Add Component modal with all configuration options

Field Descriptions

Field Required Description
Component Name Required Display name (e.g., "House Rent Allowance")
Component Code Required Unique identifier code (e.g., "HRA"). Used in reports.
Component Type Required - Earning: Added to salary
- Deduction: Subtracted from salary
Calculation Type Required - Fixed: Same amount for all
- Percentage: Calculated as % of another component
Value Required Fixed amount (e.g., 5000) or percentage value (e.g., 40)
Percentage Of For Percentage type Which component to calculate percentage of (usually Basic)
Is Taxable Optional If ON, this component is included in taxable income calculation
Is Statutory Optional If ON, marks as government-mandated (PF, ESI, etc.)
Description Optional Additional details about the component

Percentage-Based Components

When you select "Percentage" as the calculation type, additional fields appear:

Percentage calculation fields showing "Percentage Of" dropdown
💡
Common Percentage Components:
  • HRA: Usually 40-50% of Basic
  • DA (Dearness Allowance): 10-20% of Basic
  • PF (Provident Fund): 12% of Basic
  • ESI: 0.75% of Gross (for employees earning under limit)

Editing Components

Click the edit icon on any component row to modify its settings.

Edit Component modal with pre-filled values
⚠️
Impact of Changes: Editing a component affects all salary structures using it. Changes only apply to future payroll runs, not historical data.

Office Tax Rules define how each tax is calculated for specific offices. Switch to the "Office Tax Rules" sub-tab.

Office Tax Rules sub-tab showing rules by office

Calculation Types

When creating a tax rule, you can choose from three calculation methods:

Fixed Amount

A flat amount deducted regardless of salary.

Fixed amount calculation - same deduction for everyone

Percentage

Calculate as a percentage of Basic, Gross, or Taxable Income.

Percentage calculation with base selection

Slab-Based

Different amounts for different salary brackets (common for Professional Tax).

Slab-based calculation with configurable brackets
ℹ️
Slab Configuration: Use the + button to add more slabs. Each slab defines a salary range and the tax amount/percentage for that range. The system automatically applies the correct slab based on the employee's salary.

7 Arrears

Arrears handle salary differences that need to be paid retroactively, such as from promotions, appraisals, or corrections with past effective dates.

Arrears Overview

Arrears tab showing pending and processed arrears

When Arrears Are Created

  • Salary Revision: When a salary increase is effective from a past date
  • Structure Change: When a new structure version is applied retroactively
  • Correction: When a payroll error needs to be corrected
  • Promotion: When a promotion with higher salary is backdated

Table Columns

Column Description
Employee Employee name and code
Type Arrear type (Salary Revision, Correction, etc.)
Period Months for which arrears are calculated
Amount Total arrear amount
Status Pending, Approved, Paid
💡
Automatic Inclusion: Approved arrears are automatically included in the next payroll run. You can also manually add arrears during payroll processing.

10 Best Practices

Follow these recommendations for efficient payroll management.

Monthly Payroll Checklist

  1. Review Attendance
    Verify all attendance records are complete and approved. Check for any pending regularization requests.
  2. Process Leave Approvals
    Ensure all leave requests for the month are approved or rejected.
  3. Check Loan EMIs
    Verify active loans and confirm EMI amounts are correct.
  4. Review Pending Arrears
    Check for any approved arrears that need to be included.
  5. Create Payroll Draft
    Generate the payroll draft and review calculated amounts.
  6. Verify Individual Payslips
    Spot-check a few employees for calculation accuracy.
  7. Get Approval
    Have the payroll reviewed and approved by authorized personnel.
  8. Process Payment
    Export bank file and initiate salary transfers.

Common Mistakes to Avoid

⚠️
  • Processing payroll before all attendance is finalized
  • Forgetting to include pending arrears
  • Not verifying new employee salary structures
  • Approving payroll without proper review
  • Missing loan EMI deductions for approved loans
  • Not updating salary structures after appraisals

Version Control Tips

  • Create versions for policy changes: When PF rates or other statutory rates change
  • Use meaningful names: Include date or reason in version description
  • Test before applying: Use Preview feature to verify calculations
  • Document changes: Add clear notes explaining what changed and why

✓ Setup Checklist

Use this checklist to ensure your payroll system is fully configured:

  1. Create Salary Components
    Set up all earnings (Basic, HRA, DA, etc.) and deductions (PF, ESI, PT, etc.).
  2. Configure Location Taxes
    Create tax types and set up office-specific rules for each location.
  3. Create Salary Structures
    Build structure templates by combining components.
  4. Assign Structures to Employees
    Use bulk assign or individual assignment to link employees to structures.
  5. Set Up Loan Policies
    Configure loan types, interest rates, and approval workflows.
  6. Test with Preview
    Use Preview Calculation to verify your structure calculates correctly.
  7. Run First Payroll
    Create a draft, review, and process your first payroll run.
🎉
You're Ready! Once all these steps are complete, you can confidently process monthly payroll with accurate calculations and proper deductions.
