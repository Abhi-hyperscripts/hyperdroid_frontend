## SEO Checklist for Each New Page

### On-Page SEO
- [ ] Title tag (50-60 characters, keyword at start)
- [ ] Meta description (150-160 characters, includes CTA)
- [ ] H1 tag (one per page, includes primary keyword)
- [ ] H2/H3 structure (logical hierarchy)
- [ ] Keyword in first 100 words
- [ ] Internal links to product pages (3-5 per article)
- [ ] External links to authoritative sources (2-3 per article)
- [ ] Images with alt text
- [ ] URL structure (short, includes keyword)

### Schema Markup
- [ ] Article schema for blog posts
- [ ] FAQ schema for FAQ sections
- [ ] HowTo schema for guides
- [ ] SoftwareApplication schema for product pages
- [ ] BreadcrumbList schema for navigation

### Technical
- [ ] Page loads in <3 seconds
- [ ] Mobile responsive
- [ ] No broken links
- [ ] Canonical URL set
- [ ] Added to sitemap.xml
