Features

Enterprise file management made simple

A complete file storage solution with advanced upload capabilities, secure sharing, and full compliance tracking.

Chunked Uploads to 5GB

Three upload methods: direct (100MB), multipart (5GB), and chunked (10MB chunks). Resumable uploads with session tracking for unreliable networks.

Folder Organization

Hierarchical folder structure with unlimited nesting. Breadcrumb navigation, file counts, and total size tracking per folder.

Secure Sharing

Share files or folders with password protection (BCrypt), download limits, and expiry settings. Anonymous access option with share token security.

Full Audit Logging

Every action tracked: uploads, downloads, shares, deletions. IP address, user agent, and detailed JSON context for compliance and investigation.

S3-Compatible Storage

Works with AWS S3, Wasabi, MinIO, or any S3-compatible service. Presigned URLs for secure direct uploads and downloads without exposing credentials.

Real-time Updates

SignalR notifications for file uploads, deletions, and folder changes. Storage usage updates in real-time across all connected clients.
